## Project Description
This is a graphical Minesweeper game built with C++ and SFML. It features a leaderboard, pause/debug modes, and customizable grid size/mines via `config.cfg`.

## Setup Instructions
1. Install [SFML](https://www.sfml-dev.org/download.php) and ensure your compiler matches the SFML build.
2. Clone this repository (see below).
3. Open the project in CLion or your preferred C++ IDE.
4. Build and run the project.

## Usage
1. Launch the game.
2. Enter your name.
3. Play Minesweeper! Use left-click to reveal tiles, right-click to flag mines.
4. View the leaderboard after winning.

## File Structure
- `main.cpp`: Main game logic
- `files/`: Resources (images, fonts, config, leaderboard)
- `CMakeLists.txt`: Build configuration
